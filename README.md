# gestion-alumnos-java
Aplicación de consola desarrollada en Java para gestionar alumnos. Permite registrar estudiantes, almacenar sus datos en un arreglo, listar alumnos y calcular promedios utilizando Programación Orientada a Objetos.

Para fines educativos, este proyecto utiliza arreglos (Alumno[]) y una implementación manual de redimensionamiento dinámico en lugar de ArrayList. El objetivo es comprender cómo funcionan internamente las estructuras dinámicas de datos antes de utilizar las colecciones provistas por Java.
